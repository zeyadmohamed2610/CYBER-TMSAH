## المواد الدراسية

7 مواد تخصص الأمن السيبراني متاحة مع محاضرات ومراجعات لكل مادة:
- أساسيات الأمن السيبراني
- الشبكات
- أنظمة التشغيل
- الرسم الهندسي
- اللغة الإنجليزية
- التفاوض والحل الإبداعي
- تكنولوجيا المعلومات

## نظام الحضور الذكي المنيع (Anti-Cheat)

```
الدكتور أو المعيد يبدأ الجلسة
    ↓
يتم توليد جلسة حضور برقم تعريفي مخصص ومحمي
    ↓
يقوم الطالب بتأكيد هويته وربط جهازه لتسجيل الحضور
    ↓
يتم تسجيل الحضور وربط الحساب بالجهاز تلقائياً
```

**مزايا الحماية الصارمة (Zero Bypass):**
- **قفل الأجهزة (Device Fingerprint):** حساب الطالب يرتبط بجهازه الأول الذي يقوم بتسجيل الدخول منه. يمنع هذا قيام طالب بتسجيل حضور لزملائه باستخدام جهازه.
- **التسجيل المباشر اللحظي:** تسجيل تفاعلي متزامن على لوحة تحكم الدكتور/المعيد.
- **تقييد الصلاحيات:** تحكم كامل من قبل الإدارة لإلغاء قفل جهاز أي طالب في حالة تغييره للهاتف.

## لوحة التحكم

### رئيس المنصة
- 8 أقسام إدارية: المحاضرات، الجدول الدراسي، المواد، الطلاب، الدكاترة، المعيدين، الأجهزة (لإلغاء القفل)، سجلات الحضور، التسجيل اليدوي.
- إنشاء وإدارة المستخدمين بأدوار مختلفة.
- نشر وتعديل الجدول الدراسي التفاعلي.
- تصدير التقارير بصيغ متعددة بنقرة واحدة.

### الدكتور
- إنشاء المحاضرات وبدء الجلسات التفاعلية.
- متابعة الحضور لحظياً مع تحديث أوتوماتيكي.
- تصدير تقارير المحاضرات ورفع الملفات التوضيحية للمادة المخصصة.

### المعيد
- رؤية مخصصة للطلاب والسكاشن المرتبطة بمادته فقط.
- بدء جلسات حضور للسكاشن ومتابعة السجلات.

### الطالب
- عرض الجلسات النشطة للمواد.
- تسجيل الحضور بضغطة زر.
- عرض نسبة وحالة الحضور لكل مادة بشفافية.

## الأداء والاستجابة العالية (UI/UX)

- **رسوميات وهياكل متجاوبة (Responsive Grids):** تصميم مرن وجذاب يعمل على جميع الأجهزة (كمبيوتر، تابلت، هواتف صغيرة) بفضل التدرج التصميمي السلس.
- **أداء بحث ممتاز (Debouncing):** فلاتر وبحث سريع بدون التحميل الزائد على الخوادم (تأخير مدروس بـ 300ms).
- **سهولة القراءة (Accessibility):** تناسق وتطابق عالي في تباين الألوان لقراءة مريحة للعين.
- **استجابة الجداول (Scrollable Tables):** حماية الجداول وعرض البيانات التقنية من تكسير القالب الرئيسي للمنصة على الأجهزة المحمولة.
