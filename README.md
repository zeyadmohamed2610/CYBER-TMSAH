## ما تقدمه المنصة

### للطلاب
- مواد دراسية مع محاضرات ومراجعات لكل المواد
- جدول دراسي أسبوعي كامل لكل سكشن
- نظام حضور ذكي بال GPS مع كود جلسة متحرك
- تسجيل الحضور حتى بدون إنترنت (مزامنة تلقائية)
- إشعارات فورية بالامتحانات والكويزات
- قفل الجهاز لحماية الحساب

### للدكاترة والمعيدين
- إدارة جلسات الحضور والمحاضرات
- عرض الحاضرين والغياب مباشرة
- تصدير التقارير (CSV, Excel, PDF)

### لرئيس المنصة
- إدارة المستخدمين (طلاب، دكاترة، معيدين)
- إدارة الجدول الدراسي (شبكة تفاعلية)
- إدارة المواد والمحتوى
- إضافة إشعارات ومواعيد امتحانات
- قفل وإلغاء قفل أجهزة الطلاب
- سجلات النظام الكاملة
