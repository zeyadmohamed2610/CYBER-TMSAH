## نظام الحضور الذكي

```
الدكتور يبدأ الجلسة
    ↓
يظهر كود متحرك (6 أرقام) كل 60 ثانية
    ↓
الطالب يدخل الكود + يؤكد الموقع GPS
    ↓
يتم تسجيل الحضور تلقائياً
    ↓
حتى بدون إنترنت — يتم المزامنة لاحقاً
```

**حماية ضد التحايل:**
- قفل الجهاز (جهاز واحد لكل طالب)
- كود يتجدد كل دقيقة
- تحديد الموقع GPS مطلوب
- ربط بالجهاز (Device Lock)
