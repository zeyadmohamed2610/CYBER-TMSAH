- اللغة الإنجليزية
- التفاوض والحل الإبداعي
- تكنولوجيا المعلومات

## نظام الحضور الذكي المنيع (Anti-Cheat)

```
الدكتور أو المعيد يبدأ الجلسة مع تحديد النطاق الجغرافي
    ↓
يظهر كود متجدد يتغير كل 10 ثوانٍ على شاشة العرض
    ↓
الطالب يدخل الكود ليتم مطابقة الوقت والموقع الجغرافي
    ↓
يتم تسجيل الحضور تلقائياً (أونلاين أو أوفلاين)
```

**مزايا الحماية الصارمة (Zero Bypass):**
- **كود ديناميكي (TOTP):** يتغير كل 10 ثوانٍ فقط، مما يجعل تصويره أو مشاركته عبر واتساب بلا فائدة
- **النطاق الجغرافي الإلزامي (GPS):** لن يُقبل أي حضور خارج المسافة المحددة للقاعة
- **قفل الأجهزة (Device Lock):** حسابك مرتبط بجهازك الشخصي فقط
- **العمل بدون إنترنت:** تسجيل الحضور يعمل حتى بدون اتصال بالإنترنت ومزامنة لاحقة

## لوحة التحكم

### رئيس المنصة
- 9 أقسام إدارية: المحاضرات، الجدول الدراسي، المواد، المستخدمين، المعيدين، الأجهزة، التسجيل اليدوي، الإشعارات، السجلات
- إنشاء وإدارة المستخدمين بأدوار مختلفة
- نشر الجدول الدراسي التفاعلي
- إدارة المواد والمحتوى الدراسي
- تصدير التقارير بصيغ متعددة

### الدكتور
- إنشاء المحاضرات وبدء الجلسات
- عرض كود الحضور المباشر (TOTP) مع QR
- متابنة الحضور لحظياً مع تحديث تلقائي
- تصدير تقارير المحاضرات

### المعيد
- إدارة السكاشن الخاصة بالمادة المعينة
- بدء جلسات الحضور للسكاشن
- متابنة حضور الطلاب

### الطالب
- عرض الجلسات النشطة مع كود الحضور
- تسجيل الحضور بالكود أو مسح QR
- عرض نسبة الحضور لكل مادة
- إشعارات بالجلسات والامتحانات
- تسجيل الحضور حتى بدون إنترنت
